# Power Hours — Facebook Group Post Templates
**Group:** Business Marketing Mixer  
**Post day:** Monday (day before the session)  
**Goal:** Drive registrations to knowhowmarketinglab.com/power-hours  
**Tone:** Conversational, helpful, no geekspeak  

---

## How to use these templates

1. Pick one of the 5 hooks below — rotate them weekly so the group doesn't see the same post twice in a row.
2. Pair it with the matching social image (see image column).
3. Swap in the actual date before posting.
4. Post on **Monday morning** — aim for 8–9am Pacific for best reach.
5. Pin the post on Tuesday morning with the Zoom link added as a comment.
