# GHL Notification Setup — KnowHow Marketing Lab

Set up email notifications in GoHighLevel so both Pip and Phelan receive an alert whenever someone submits a form or books a call. This takes about 5 minutes across all five touchpoints.

**Recipients for all notifications:**
- pip@seymourdigitalmedia.com
- phelan@seymourdigitalmedia.com

---

## 1. Power Hours Registration Form

The Power Hours registration form is embedded on `knowhowmarketinglab.com/power-hours`.

1. In GHL, go to **Sites → Forms**
2. Find the form named **"Free Power Hours"** (ID: `VpNFCGnnrKnymB81G7bB`)
3. Click the three-dot menu → **Edit**
4. Go to the **Notifications** tab (or **Settings → Notifications**)
5. Under **Email Notifications**, click **Add Notification**
6. Set **To:** `pip@seymourdigitalmedia.com` — Subject: `New Power Hours Registration`
7. Add a second notification for `phelan@seymourdigitalmedia.com`
8. Save

---

## 2. Free Course Opt-In Form

The free course opt-in form appears in the LeadCapture section of the home page.

1. In GHL, go to **Sites → Forms**
2. Find the form named **"KnowHow Free Course Opt In"** (ID: `l71tHQPMO7st6TFSPmo5`)
3. Click the three-dot menu → **Edit**
4. Go to **Notifications** → **Add Notification**
5. Set **To:** `pip@seymourdigitalmedia.com` — Subject: `New Free Course Sign-Up`
6. Add a second notification for `phelan@seymourdigitalmedia.com`
7. Save

---

## 3. SEO & Google Ads Coaching Booking

This calendar is embedded on the `/courses/seo` and `/courses/google-ads` pages.

1. In GHL, go to **Calendars**
2. Find the calendar linked to booking ID `vRBWL3TL9wPMAoiDTQuH`
3. Click **Edit** (pencil icon)
4. Go to the **Notifications** tab
5. Under **Team Notifications**, add:
   - `pip@seymourdigitalmedia.com` — trigger: **New Booking** and **Cancellation**
   - `phelan@seymourdigitalmedia.com` — trigger: **New Booking** and **Cancellation**
6. Save

---
