# Google Ads Campaign Plan — KnowHow Marketing Lab

**Business:** KnowHow Marketing Lab | knowhowmarketinglab.com
**Location:** British Columbia, Canada (target: Canada + USA)
**Goals:** Drive sign-ups for The Lab ($29/mo) and the Free AI + SEO Course
**Budget recommendation:** Start at $10–15/day CAD (~$300–450/mo) and scale based on cost-per-lead
**Account type:** Standard paid Google Ads (not Ad Grant)

---

## Campaign Architecture

Run **3 campaigns** targeting different intent levels:

| Campaign | Goal | Daily Budget | Landing Page |

| 3. Power Hours | Free registrations (top of funnel) | $3/day | `/power-hours` |

## Conversion Tracking Setup (Do This First)

Before launching ads, set up these conversion actions in Google Ads:

| Conversion | Type | How to Track |
|---|---|---|
| Free course sign-up | Lead | GHL form submission → fires on `/thank-you` page load |
| Power Hours registration | Lead | GHL form submission → fires on `/thank-you/power-hours` page load |
| The Lab join | Purchase | GHL/GoHighLevel checkout → fires on confirmation page |

**Recommended:** Use Google Tag Manager to fire conversion tags. The site already has GTM installed (GTM-W449WJS).

## Pre-Launch Checklist

- [ ] Google Ads account created at ads.google.com
- [ ] Billing set up (CAD)
- [ ] GTM conversion tags set up for all 3 conversion actions
- [ ] Google Ads linked to Google Analytics 4 (GA4)
- [ ] All 3 campaigns created with correct landing pages
- [ ] Negative keyword list added (jobs, career, free download, youtube, reddit, etc.)
- [ ] Sitelink and callout extensions added
- [ ] Geo targeting set to Canada (BC priority)
- [ ] Conversion tracking verified with Tag Assistant
- [ ] First week: check search terms report daily and add negatives

---

## Monthly Management Tasks (Once Live)

1. Review search terms report — add negatives weekly for first month
2. Check Quality Score on all keywords (must be 3+ to avoid wasted spend)
3. Review ad strength in RSA — aim for "Good" or "Excellent"
4. Pause any keyword with 100+ clicks and 0 conversions
5. Test new headlines monthly — rotate in fresh copy
6. Check landing page conversion rate in GA4 (target: 15%+ for free offers)
7. Review impression share — if below 50%, increase bids or budget
